- 학교에 전화할 일이 있는데 번호를 찾기 힘들어서 곤란했던 적이 있지 않으신가요?

  ex) 번호를 몰라 대표전화로 걸어보신 분 !!🤚

      동동: 저 ooo학과 사무실 번호 알 수 있을까요?
